This project is a really basic recreation of the famous Wolfenstain 3D game. This is an university project consisting of implementing the ray casting technique for visualisation and image recreation. The project is developed using Python 3 and Pygame graphics library.
1 For shadows you have to comment lines (90-93 and 154-158) and reactivate the function dessine_mur(lignes 87 et 151) and  distmax(ligne 74) that are commented in the main module - main.py.  

2. As this project was assigned by an university in France, most of the comments,function names, variable names and docstrings are in French.

3. You quit with ESC.
